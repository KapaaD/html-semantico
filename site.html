O que são Tags Semânticas?

As Tags Semânticas do HTML são elementos que conferem significado ao conteúdo, premitindo uma melhor compreensão tanto para os desenvolvedores qunato para os mecanismos de busca.

Elas ajudam a estruturar o conteúdo de forma mais clara e organizada, contribuindo para àcessibilidade e usabilidade do site.

Por que devemos usar o HTML Semântico?

Ao escolher não ultilizar o HTML Semântico ele fica confuso, alem de ter um ranqueamendo pior nas ferramentas de navegação, fazendo com que o seu site seja recomentado para menos pessoas.

Como usar Tags Semânticas

Usar Tags Semânticas não muito dificil, é só usar a Tag respectiva ao significado do documento, alguns exemplos são <header>, <dev>, <main>, <fooder>, <section>, <nav> etc...
